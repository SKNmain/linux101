1. Sprawdzenie i zmiana uprawnień

ls

ls -la/ls -l plik

Argument Opis
-l Wyświetl w postaci listy ze szczegółami
-a Wyświetl wszystkie pliki (także ukryte)

chmod

chmod kod plik Kod się sumuje, np. 7 jest połączeniem praw rwx. Kod składa się z trzech cyfr, w kolejności uprawnienia: user grupa all, np. 744.

Argument Opis
kod Wartość uprawnień
r=4 read(odczyt)
w=2 write(zapis)
x=1 execute(uruchamianie)

5. Znajdowanie duplikatów

find . ! -empty -type f -exec md5sum {} + | sort | uniq -w32 -dD
Argument Opis
! Operator not
-empty Puste pliki
-type f Tylko pliki
-exec md5sum {} + Oblicza sumę kontrolną dla tych plików
sort Sortuje linie alfabetycznie
uniq Wyświetla unikalne linie (bez argumentów wyświetla je tylko raz)
-wx Bierze pod uwagę tylko x pierwszych znaków
-d -D Wyświetlanie tylko duplikatów i wszystkie duplikaty
